## Client Installation

```bash
$ cd client 
$ yarn 
$ cd .. 
```

## Server Installation

```bash
$ cd server 
$ yarn 
$ cd .. 
```


## Client run

```bash
$ cd client 
# development
$ yarn dev
$ cd .. 
```

## Server run

```bash
$ cd server 
# development
$ yarn start:dev
$ cd .. 
```

## Docker

```bash
$ docker compose up
```
